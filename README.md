# 🐍 Fundamentos Python

Repositório com exercícios e estudos práticos dos fundamentos da linguagem Python, organizado por temas para facilitar a evolução do aprendizado e a navegação no portfólio.

## 📚 Estrutura do repositório

### 01-logica
Exercícios de lógica de programação, estruturas condicionais, operadores relacionais e repetição.

### 02-funcoes
Exercícios com criação de funções, reutilização de código, organização da lógica e tratamento de entradas.

### 03-listas-tuplas-dicionarios
Prática com estruturas de dados em Python, incluindo listas, tuplas e dicionários.

### 04-calculos
Exercícios voltados para cálculos, conversões e operações matemáticas aplicadas.

### 05-programacao-orientada-a-objetos
Estudos e práticas de Programação Orientada a Objetos (POO) com Python.

### 06-interfaces
Exercícios com interface gráfica utilizando Tkinter.

## 🎯 Objetivo

Consolidar a base em Python para avançar de forma estruturada nas áreas de:

- Análise de Dados
- Automação com Python
- Backend com Python

## Tecnologias utilizadas

- Python

## Observação

Este repositório reúne estudos práticos e exercícios desenvolvidos durante minha formação e prática contínua com Python.
